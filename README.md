# Jekyll based blog: [blog.kaffeekantate.xyz](https://blog.kaffeekantate.xyz/)

## [Jekyll Commands](https://jekyllrb.com/docs/usage/)

### Build

```shell
$ jekyll build
```

### Run development server with watch

```shell
$ jekyll serve
```

## Rake Tasks

### Initial setup

```shell
$ rake setup
```

### Deploy to gh-pages

```shell
$ rake deploy
```
